Backend Architecture
1. API Layer

Handles:
        new game
        player move
        AI move request
        game state retrieval

Example routes:
        POST /game/start
        POST /game/move
        POST /game/ai-move
        GET  /game/state

2. Game Engine Layer

This is the core logic.

Responsibilities
    board initialization
    legal move generation
    captures
    king promotion
    turn management  
    terminal state detection
